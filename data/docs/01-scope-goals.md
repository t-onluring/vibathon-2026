# Product Scope & Goals

## Goal
Membangun Layer-1 registry sumber kajian sunnah + monitoring health otomatis (Phase 1: Telegram).

## Target User
- Maintainer source list
- Aggregator kajian
- Developer yang konsumsi dataset

## In Scope (Saat Ini)
- Registry source berbasis JSON di Git
- Telegram health checker harian
- Dashboard publik (Plan & Roadmap + Live Dashboard)
- Open contribution via PR

## Out of Scope (Saat Ini)
- Auth/user accounts
- Database eksternal
- Scraping IG/FB di Phase 1
- End-user personalization
