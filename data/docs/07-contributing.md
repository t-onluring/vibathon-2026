# Contributing Guide

Panduan kontribusi resmi ada di:
- `docs/CONTRIBUTING.md`

## Ringkas
1. Buat issue (Source Add/Update)
2. Fork + branch `contrib/<ringkas-perubahan>`
3. Edit `data/sources.json`
4. Jalankan:
   - `npm run validate:sources`
   - `npm run lint`
   - `npm run build`
5. Buka PR + evidence
