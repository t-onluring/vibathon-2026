# Quick Start

## Prasyarat
- Node.js 20+
- npm

## Jalankan Lokal
```bash
npm install
npm run dev
```
Buka `http://localhost:3000`.

## Command Penting
```bash
npm run check:telegram
npm run validate:sources
npm run lint
npm run build
npm run start
```

## Data & Docs
- Source registry: `data/sources.json`
- Snapshot latest: `data/latest.json`
- Arsip health: `data/health/`
- Docs yang dirender app: `data/docs/`
