Data Model

Three new optional fields on the Source interface enable hierarchical relationships while maintaining full backward compatibility with existing flat entries.

The source_type field distinguishes channels, groups, and topics. Parent-child linkage uses parent_id and topic_id on child entries. Group scores are computed as max(own_freshness, avg(children_scores)) with zero-child fallback.

export type SourceType = "channel" | "group" | "topic"; export interface Source { // ... existing fields unchanged ... source_type?: SourceType; parent_id?: string; topic_id?: string; } export function isTopicSource(s: Source): s is Source & { source_type: "topic"; parent_id: string; topic_id: string; } { return s.source_type === "topic" && !!s.parent_id && !!s.topic_id; }
