Event Schema

The KajianEvent interface defines the structured output produced by the text extraction pipeline.

interface KajianEvent {
  id: string;
  source_id: string;
  source_message_id?: number;

  // Core
  title?: string;
  ustadz: string;
  tema?: string;

  // When
  date: string;           // YYYY-MM-DD
  time_start?: string;    // HH:mm (WIB)
  time_end?: string;
  hijri_date?: string;

  // Where
  masjid: string;
  address?: string;
  city: string;
  geolocation?: {
    lat: number;
    lng: number;
    gmaps_url: string;
  };

  // Meta
  format: "rutin" | "tabligh-akbar" | "daurah" | "online";
  gender?: "ikhwan" | "akhwat" | "umum";
  contact?: string;
  extracted_from: "text" | "image" | "video";
  extraction_confidence: number;  // 0-1
  raw_text?: string;
  extracted_at: string;
  verified: boolean;
}
03

Pipeline Steps

Step 1: Telegram Bot API integration
Create bot via @BotFather, add to @sijadwalkajian group (read-only), implement polling or webhook.
scripts/ingest/telegram-bot.ts
Step 2: Message ingestion pipeline
Poll messages from monitored topics. Store raw messages in data/messages/ (JSON, keyed by date). Dedup by message_id.
scripts/ingest/poll-messages.ts
Step 3: Text extraction with LLM
Send plain text messages to Claude/GPT-4o with structured output schema. Extract kajian events with confidence scores.
scripts/extract/text-extractor.ts
Step 4: Geolocation resolver
Parse goo.gl/maps/ and maps.app.goo.gl/ short URLs. Resolve to lat/lng coordinates. Cache results.
scripts/extract/geo-resolver.ts
Step 5: Event storage and dedup
Store extracted events in data/events/ (JSON, keyed by date). Dedup by composite key: source_id + date + masjid + ustadz.
scripts/extract/store-events.ts
Step 6: Static API generation
Generate pre-built JSON files from sources + health + events data at public/api/v1/.
scripts/generate-api.ts
Step 7: GitHub Actions workflow
Cron schedule every 6 hours: poll messages, extract events, generate API, commit.
.github/workflows/ingest-events.yml
Step 8: Event display in dashboard
New tab in Live Dashboard showing upcoming kajian events. Filter by region, date, ustadz.
src/app/components/EventsTab.tsx
